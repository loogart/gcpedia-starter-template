GCpédicure

Le remède aux pages web ennuyantes de GCpédia.
Une pédicure pour leur amener du style.

GCpédia manque un peu de pep?

Croyez-vous que GCpédia limite votre marque visuelle? Voulez-vous que votre présence GCpédia soit mémorable à travers le gouvernement canadien? Voici la solution que vous cherchiez: GCpédicure.

As-tu du css?

Vous avez certainement remarqué quelques sites web intéressants: l'InnoLab, le programme pilote de l'analyse de données, le salon de l'innovation 2017 du BCP et le site GC-Exp. Tous ces sites web ont quelque chose en commun: ils utilisent tout l'espace de votre navigateur web et ressemblent à un site web moderne. Ils utilisent tous la formule secrète GCpédicure.

Voulez-vous créer quelque chose de similaire?

Suivez les instructions ci-bas et vous devriez être en mesure de bâtir un site web GCpédicure similaire sur GCpédia. Notez que vous aurez besoin des notions de base de l'html pour réussir, donc si vous cherchez à en apprendre plus sur ce sujet, visitez w3schools.com et commencez.

Une étape à la fois, une ligne à la fois...

Commencez ici. Vous pouvez visionner les détails de chaque étape en cliquant sur celle-ci.

1. Téléchargez le modèle de base

Téléchargez le modèle de base de GCpédicure. Extraire le contenu et ouvrez le fichier Notepad. Copiez tout le texte dans le fichier Notepad (CTRL + A, then CTRL + C).

Télécharger

2. Créez votre nouvelle page GCpédia

Ouvrez un nouvel onglet dans votre navigateur, accédez à GCpédia et appuyez sur Connectez-vous en haut à droite. Si vous n'avez pas de compte, créez-le en suivant les instructions à l'écran - sinon, connectez-vous tout simplement.

Dans la zone "Recherchez GCpédia" (note: pas la "recherche GCTools" en haut à droite, celle juste en-dessous), tapez le nom de la nouvelle page que vous souhaitez créer. Par exemple: "Ma page".

Si le nom de la page est déjà pris, essayez un autre nom et cliquez sur Rechercher. Si la page n'existe pas, allez-y et créez la page "Ma page".

Une fois votre nouvelle page créée, copiez l'URL actuelle et enregistrez-la quelque part (bloc-notes, document Word, ...). C'est extrêmement important parce que vous allez devoir modifier votre page dans un futur proche.

Maintenez coller (CTRL + V) le contenu que vous avez copié dans la nouvelle boîte d'éditeur de texte.

Une fois que tout le contenu est collé, appuyez sur CTRL + F et tapez "###". Sous le ###, vous verrez une ligne qui commence avec <li><a href="...>. Copiez l'URL que vous avez enregistrée plus tôt et remplacez tout le texte existant à l'intérieur guillemets par l'URL enregistrée. Maintenant, vous appuyez sur le bouton "Enregistrer la page" au bas de la page.

Voilàààààà! Vous avez maintenant réussi à configurer une nouvelle page modifiable avec notre modèle GCpédicure. Prochaine étape, modifier cette page.

3. Modifiez cette page modèle

C'est ici que vous aurez besoin de vos connaissances html. Cette page modèle est bâtie sur le système html Bootstrap. Si vous avez de l'expérience avec Bootstrap, vous pouvez maintenant faire de la magie en utilisant les composantes que Bootstrap offre. Pour ajouter votre css personnalisé, vous pouvez soit lier une page GCpédia séparée en incluant {{le nom de votre page css}} au bas de votre code ou tout simplement en créant votre css wà l'intérieur des crochets <style></style> dans la section head de votre code.

Si vous n'avez aucune expérience avec l'html, cherchez de l'aide d'un collègue qui peut faciliter ce processus (*). Ou commencez à apprendre les bases de l'html au site w3schools.com.

*Note: L'InnoLab ne s'implique pas dans la création ou la personnalisation de votre page GCpédia, mais nous sommes là pour vous aider et répondre à vos question.
